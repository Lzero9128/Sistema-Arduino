Información

1Conexión

Se muestra el esquema de conexión donde estan dos modalidades para la pantalla LCD 20X4 mostrando tambien pines donde se conecta cada uno de los sensores utilizados.

2Pantallas LCD

Las pantallas mostradas en el esquema muestran la configuracion tanto con el modulo i2c y sin este modulo ya que de ambas formas se puede conectar y mostrar los datos.

3Modulo w5100 y Arduino Mega

El Modulo w5100 se moenta sobre el Arduino Mega haciendo coincidir los pines ISCP, posteriormente se programa la tarjeta Arduino de forma habitual incluyendo la libreria Ethernet.h para que se active el modulo w5100 y trabaje en conjunto con el Arduino.
